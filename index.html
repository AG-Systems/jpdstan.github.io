  • programming

  • Cyanogen

    I spent my summer of 2016 interning at Cyanogen, Inc., the private branch of CyanogenMod which develops an open source Android alternative. More details can be found in my resume, but in summary I worked with the Services team and built a lightweight service for push notifications. I built it with Vert.x, an application framework running on the JVM featuring an async and non-blocking HTTP and I/O API. Apache Cassandra served as the database layer.

  • Ryde

    Ryde is a very early-stage startup in auto care. I developed the MVP of the web platform of their application, "Ryde", using Angular to create the UI and Firebase for the datastore. I also built our Node/Express backend to migrate our logic away from the client and to integrate external services into our application. Stripe and Braintree are examples of these services, both of which I implemented on our application to process payments. So far we've processed over $12k in just a month!

  • Gitlet

    One of my more notable school projects. Gitlet is a version control system for managing products, mirroring the features of Git. Emphasized the serialization and manipulation of files to compress, access, and manage file content efficiently. Stressed the employment of appropriate data structures. Code available on request.

  • Fastext

    Fastext is a simple speed reading tool, built for the purpose of practicing some JavaScript and jQuery.

    Fastext flashes words based on a selected speed (WPM), utilizing cognitive potential to read more words in a shorter period of time. We typically read words in our head before interpreting them, which severely limits our speed as our brain is much faster at interpreting words than we are at saying them in our head.

    A website I made for my high school robotics team in 2014. Unfortunately, the domain now points to a new website managed by some lame CMS, but the source code for the original website still exists, and you can view the website on your own server.
